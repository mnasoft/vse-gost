
* Введение
Проект *vse-gost* предназначен для разбора информации о ГОСТ, размещенной на
сайте [[http://vsegost.com]].

Здесь приведена [[./org/catalog.org][структура каталогов]] сайта vsegost.com.

* Проект *vse-gost* содержит инструкции по:
1. [[./org/instruction-mirroring.org][Зеркалированию сайта vse-gost.com]]
2. [[./org/instruction-import-file.org][Генерированию файла импорта для ДБ PostgreSQL]]
3. [[./org/instruction-postgesql.org][Установке и настройке PostgreSql]]
4. [[./org/instruction-gost.org][Созданию базы данных gost]]
5. [[./org/instruction-build-web-app.org][Сборке выполняемого файла, запускающего веб-сервер]]
6. [[file:org/instruction-start-web-app.org][Запуску веб приложения]]

** Последующие действия
См. проект vse-gost/web.

* Запуск сервера
В отдельной консоли необходимо выполнить
#+begin_src shell
  vsegost-web-tmux.sh
#+end_src

* Standalone сборка/установка (Linux)
Для standalone используется SBCL-образ с точкой входа
=vse-gost/web:start-gosts=.

Скрипты находятся в каталоге =src/web/sh=.

** Сборка + установка (рекомендуется)
#+begin_src shell
  cd src/web/sh
  ./build-install-standalone.sh build
  ./build-install-standalone.sh install
#+end_src

Без аргументов скрипт показывает справку:
#+begin_src shell
  ./build-install-standalone.sh
#+end_src

Будут установлены команды в =/usr/local/bin=:
- =vsegost-web=           (standalone бинарник)
- =vsegost-web.sh=        (обертка запуска)
- =vsegost-web-tmux.sh=   (запуск в tmux)

** Только сборка
#+begin_src shell
  cd src/web/sh
  ./build-install-standalone.sh build
#+end_src

Артефакты сборки появляются в =src/web/sh/build/=.

** Только установка
#+begin_src shell
  cd src/web/sh
  ./build-install-standalone.sh install
#+end_src

** Удаление standalone из /usr/local/bin
#+begin_src shell
  cd src/web/sh
  ./build-install-standalone.sh uninstall
#+end_src

** Очистка артефактов сборки
#+begin_src shell
  cd src/web/sh
  ./build-install-standalone.sh clean
#+end_src

** Запуск standalone
Обычный запуск:
#+begin_src shell
  vsegost-web.sh
#+end_src

Запуск в tmux:
#+begin_src shell
  vsegost-web-tmux.sh
#+end_src

Опционально можно задать имя tmux-сессии:
#+begin_src shell
  SESSION_NAME=vsegost-web vsegost-web-tmux.sh
#+end_src
